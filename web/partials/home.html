A Google App Engine seed with Jersey for REST, custom Bootstrap styling, and AngularJS


This project serves as a starting point for web applications built for Google App Engine. It integrates GAE with Jersey for RESTFul webservices, and AngularJS for client side MVC. The Angular app is bundled with Angular-UI, Angular-Strap, Bootstrap, and jQuery, along with examples of angular services, directives, controllers, and filters.
